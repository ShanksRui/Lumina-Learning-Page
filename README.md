# Lumina Learning Page

Landing page estática para o projeto **Lumina Learning**, uma representação de uma plataforma educacional com foco em apresentação de planos, cursos

## Sobre o projeto

Este projeto foi criado como uma página web simples (HTML + CSS), com seções típicas de uma landing page:

- cabeçalho com navegação;
- área principal com título e busca;
- lista lateral de temas;
- cards de planos mensais;
- seção com exemplos de cursos;
- seção "Sobre";
- formulário de contato;
- rodapé com ícones de redes sociais.

#  Funcionalidades atuais

- Navegação por âncoras no menu.
- Layout responsivo básico para telas menores.
- Campo de busca visual (sem lógica JavaScript ainda).
- Formulário de contato estático (sem envio para backend).
